HTML

Для стандартизации вывода текста внутри браузера был придуман стандарт HTML, описывающий правила оформления текстовых данных для корректного вывода. Этот курс посвящён азам HTML разметки, работы с типографикой, вывода медиа документов и работы с формами

Где используется язык разметки HTML?

HTML – это фундаментальный язык разметки, на котором построены все веб-сайты в мире. С его помощью ты создашь структуру веб-страниц, добавишь текст, изображения, видео и формы. HTML легко освоить, он востребован в веб-разработке, а его сочетание с CSS и JavaScript поможет тебе создавать современные и удобные сайты

Чему вы научитесь на курсе “Основы HTML”

Разберётесь в базовом синтаксисе HTML — это фундаментальный навык, который позволит вам создавать структуру веб-страниц и правильно размечать контент. После курса вы сможете сверстать простую веб-страницу с текстом, изображениями, таблицами и формами
