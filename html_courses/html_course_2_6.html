HTML: Таблицы

Таблицы — один из самых привычных способов представления информации. Мы встречаем их повсюду: в документах, таблицах умножения, описаниях товаров. Такой формат позволяет удобно сравнивать характеристики различных объектов. В веб-разработке таблицы также широко используются, хотя их разметка может показаться немного сложной.

Таблица представляет собой составной элемент, который создаётся с помощью нескольких вложенных тегов. Это похоже на структуру списков, где есть общий контейнер и вложенные в него элементы.

Разметка таблицы включает в себя несколько основных элементов:

  • <table> — область таблицы.
  • <tr> — строка таблицы.
  • <thead> — шапка таблицы.
  • <th> — ячейка шапки.
  • <tbody> — тело таблицы.
  • <td> — ячейка таблицы.

Любая таблица начинается с парного тега <table>.

<table>
  <!-- Здесь будут данные таблицы -->
</table>

Внутри этого тега создаются строки и столбцы таблицы. Для добавления строк используется тег <tr>, а для создания ячеек — тег <td>. Давайте создадим таблицу с двумя строками и тремя ячейками в каждой:

<table>
  <tr> <!-- Строка -->
    <td>Ликёр «Jägermeister»</td> <!-- Ячейка -->
    <td>700 грамм</td>
    <td>12000 тенге</td>
  </tr>

  <tr> <!-- Строка -->
    <td>Виски «Jack Daniels»</td> <!-- Ячейка -->
    <td>500 грамм</td>
    <td>10000 рублей</td>
  </tr>
</table>

  
 
    Ликёр «Jägermeister» 
    700 грамм
    12000 тенге
  


  
 
    Виски «Jack Daniels» 
    500 грамм
    10000 тенге

Важно: Количество ячеек в каждой строке таблицы должно быть одинаковым.

Попробуйте вставить этот код в редактор. Обратите внимание, что визуально таблица будет выглядеть иначе, чем в примерах теории. Это связано с тем, что браузеры по умолчанию не отображают видимые границы ячеек. Для добавления границ и стилизации таблицы используется CSS — специальный язык стилей, с которым вы познакомитесь в другом курсе.

Сейчас данные таблицы отображаются, но не хватает заголовков, которые описывают содержимое каждого столбца. Для создания заголовков таблицы используется тег <thead>, а для ячеек заголовков — тег <th>. При этом структура таблицы остаётся неизменной. Давайте добавим заголовки в таблицу:

<table>
  <thead> <!-- Шапка таблицы -->
    <tr>
      <th>Вкусняшка</th> <!-- Ячейка-заголовок -->
      <th>Вес</th>
      <th>Цена</th>
    </tr>
  </thead>

  <tr> <!-- Строка -->
    <td>Ликёр «Jägermeister»</td> <!-- Ячейка -->
    <td>700 грамм</td>
    <td>12000 тенге</td>
  </tr>

  <tr> <!-- Строка -->
    <td>Виски «Jack Daniels»</td> <!-- Ячейка -->
    <td>500 грамм</td>
    <td>10000 рублей</td>
  </tr>
</table>

Заключительным шагом является добавление тега <tbody>, который используется для разметки основной части таблицы. Многие разработчики пропускают его, так как браузеры автоматически оборачивают строки, не заключённые в другие теги, в <tbody>. Обычно в таблице используется только один <tbody>. В больших таблицах можно добавлять несколько <tbody> для разделения секций, но в большинстве случаев это не требуется.

<table>
  <thead>
    <tr>
      <th>Вкусняшка</th>
      <th>Вес</th>
      <th>Цена</th>
    </tr>
  </thead>

  <tbody> <!-- Тело таблицы -->
    <tr>
      <td>Шоколад «Тота»</td>
      <td>100 грамм</td>
      <td>200 рублей</td>
    </tr>

    <tr>
      <td>Шоколад «Тота Белый»</td>
      <td>150 грамм</td>
      <td>500 рублей</td>
    </tr>
  </tbody>
</table>

Дополнительно можно добавить «подвал» таблицы с помощью тега <tfoot>. В нём можно разместить итоговую информацию, например, общую стоимость всех товаров.

Также у таблицы можно задать заголовок, который помогает различать несколько таблиц на одной странице. Это особенно полезно, если выводится несколько таблиц одновременно. Для создания заголовка используется парный тег <caption>. Заголовок должен располагаться сразу после тега <table>. Давайте добавим заголовок в таблицу:

<table>
  <caption>Ассортимент шоколада «Каки-тошки»</caption>
  <thead>
    <tr>
      <th>Вкусняшка</th>
      <th>Вес</th>
      <th>Цена</th>
    </tr>
  </thead>

  <tbody> <!-- Тело таблицы -->
    <tr>
      <td>Шоколад «Тота»</td>
      <td>100 грамм</td>
      <td>200 рублей</td>
    </tr>

    <tr>
      <td>Шоколад «Тота Белый»</td>
      <td>150 грамм</td>
      <td>500 рублей</td>
    </tr>
  </tbody>
</table>
