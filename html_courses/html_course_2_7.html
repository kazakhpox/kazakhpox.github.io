Объединение ячеек внутри строк и столбцов

В таблицах часто возникает необходимость объединять строки или столбцы, чтобы упростить представление данных. Это позволяет избежать дублирования информации. Например, в складской таблице можно объединить ячейки и указать, что товара нет на складе, вместо повторения этой информации в каждой строке.


        
          

            Товар
            Количество
            Статус
          

        
        
          

            Шоколад «Кака-Тошка»
            0
            Нет на складе 
          

          

            Шоколад «Картошка»
            0

В HTML для объединения ячеек в строках или столбцах используются специальные атрибуты colspan и rowspan. Значение атрибута указывает количество ячеек, которые нужно объединить: для colspan — ячейки справа, а для rowspan — ячейки снизу. Объединение начинается с текущей ячейки, к которой применён атрибут. Например, если указано colspan="2", то текущая ячейка объединяется с соседней справа.

Разметка для примера выше без объединения ячеек будет выглядеть следующим образом:

<table border="1" cellspacing="0" cellpadding="5" style="border-collapse: collapse; width: 100%;">
  <thead>
    <tr>
      <th>Товар</th>
      <th>Количество</th>
      <th>Статус</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Шоколад «Кака-Тошка»</td>
      <td>10</td>
      <td rowspan="2">Нет на складе</td> <!-- Объединение двух строк -->
    </tr>
    <tr>
      <td>Шоколад «Картошка»</td>
      <td>0</td>
    </tr>
  </tbody>
</table>

Чтобы объединить ячейки, необходимо выполнить два шага:

  1. Добавить в тег ячейки, которую нужно объединить, атрибут colspan со значением, равным количеству объединяемых ячеек справа.
  2. Удалить из строки лишние ячейки, которые объединяются.
<table>
<thead>
  <tr>
    <th>Товар</th>
    <th>Цена</th>
    <th>Количество</th>
  </tr>
</thead>

<tbody>
  <tr>
    <td>Чай</td>
    <td colspan="2">Нет на складе</td> <!-- Объединение двух ячеек -->
  </tr>
</tbody>
</table>

Для объединения ячеек по вертикали используется атрибут rowspan. Процесс аналогичен объединению по горизонтали, с той лишь разницей, что удалять ячейки необходимо в строках, расположенных ниже текущей.

<table border="1" cellspacing="0" cellpadding="5" style="border-collapse: collapse; width: 100%;">
<thead>
  <tr>
    <th>Сотрудник</th>
    <th>Зарплата</th>
    <th>Бонусы</th>
    <th>Менеджер</th>
  </tr>
</thead>

<tbody>
  <tr>
    <td>Дархан Редакторович</td>
    <td>750$</td>
    <td>63$</td>
    <td rowspan="2">Малика Эштемэлевна</td> <!-- Объединение двух строк -->
  </tr>
  <tr>
    <td>Вадик Простовадик</td>
    <td>1200$</td>
    <td>0</td>
  </tr>
</tbody>
</table>

В данном примере оба сотрудника работают с одним менеджером. Вместо того чтобы указывать его дважды, удобнее объединить ячейки по вертикали. Для этого использован атрибут rowspan="2", а четвёртая ячейка во второй строке была удалена, так как её заменяет объединённая ячейка из первой строки.


    
      

        Сотрудник
        Зарплата
        Бонусы
        Менеджер
      

    
    
    
      

        Дархан Редакторович
        750$
        63$
        Малика Эштемэлевна 
      

      

        Вадик Простовадик
        1200$
        0

Задание

Создайте таблицу с двумя строками и тремя столбцами. Объедините первые две ячейки во второй строке с помощью атрибута colspan. Первая строка должна выступать в роли заголовка таблицы.

Полезное

  • После объединения ячеек общее количество ячеек в каждой строке таблицы должно оставаться одинаковым. Важно понимать разницу между количеством тегов <td> и фактическим количеством логических ячеек. Например, если вы применяете атрибут colspan="2", то создаётся две логические ячейки, объединённые в одну. Поэтому необходимо удалить одну физическую ячейку из разметки, чтобы сохранить баланс.
  • При использовании атрибута colspan удаляются ячейки из той же строки. Если применяется атрибут rowspan, то удаляются ячейки из строк, расположенных ниже.
