HTML разметка

Каждый день мы сталкиваемся с множеством веб-страниц, на которых читаем тексты, оформленные как в печатных изданиях. Эти страницы являются частью гипертекста — сети взаимосвязанных документов, переход между которыми осуществляется по ссылкам. Для создания такой структуры используется язык HTML.

HTML (HyperText Markup Language) — это язык, предназначенный для разметки гипертекста. Он задаёт правила, по которым браузер распознаёт элементы страницы: заголовки, списки, таблицы, изображения и другие. Язык появился в 1993 году и стал стандартом для отображения текстового контента в интернете.

Даже эта веб-страница создана с помощью HTML и отображается браузером на основе его разметки. Несмотря на развитие технологий, HTML по-прежнему остаётся основой для структурирования веб-текста.

Пример HTML-кода:

<section>
        <h2>HTML-разметка</h2>
    <p>
        Мой курс —  это бесплатные практические уроки по 
        web-программированию для тех, кто начинает с нуля.
    </p>
    <p>
        Если вы совсем новичок и хотите научиться основам,
        этот курс вам отлично подойдет.
    </p>
</section>

Не пугайтесь такого обилия непонятных слов и знаков — к концу курса вы будете понимать работу приведённой выше разметки, а также научитесь использовать разметку на своих страницах. Обратите внимание, что вся суть в разметке выше — это текст. Остальное — вспомогательные элементы для того, чтобы вывести текст там, где было задумано дизайнерами и/или разработчиками.
