Атрибуты HTML

Разметка информации на странице содержит множество деталей и особенностей. Например, можно не только определять тип элемента, но и добавлять дополнительные данные, которые будут обработаны браузером.

Наведите курсор на текст ниже:

Наведите курсор мыши на эту строку текста.

При наведении курсора на элемент появилось небольшое окно с текстом «Всплывающая подсказка. Здесь можно указать дополнительную информацию». Такое поведение задаётся с помощью атрибутов — специальных параметров, которые влияют на отображение информации на странице. Каждый HTML-тег имеет несколько стандартных атрибутов, которые записываются в формате атрибут="значение", где:

  • Атрибут — название атрибута, например, title
  • Значение — значение атрибута, например, Всплывающая подсказка. Здесь можно указать дополнительную информацию. Оно может быть разным, в зависимости от самого атрибута.

Разметка примера со всплывающей подсказкой:

<p title="Всплывающая подсказка. Здесь можно указать дополнительную информацию">Наведите курсор мыши на эту строку текста. Лучше сделать это с компьютера</p>

Атрибут title можно применять не только к параграфам, но и к любым элементам на странице. Такие атрибуты называются глобальными, поскольку они не зависят от конкретного элемента. В отличие от них существуют атрибуты, предназначенные только для определённых тегов. Например, атрибут colspan, который используется для объединения ячеек в таблицах. Применение этого атрибута вне таблиц не имеет смысла.

Атрибуты могут косвенно влиять на отображение информации. Например, в языке стилей CSS часто используется атрибут class, который позволяет различать схожие элементы с помощью его значения. Подробнее о работе классов вы узнаете в курсе по CSS, а пока важно понимать, что сам по себе этот атрибут не влияет на вывод информации на страницу.

<p class="lead">Параграф с классом lead</p>
<p class="mega-paragraph">Параграф с классом mega-paragraph</p>

Один тег может содержать сразу несколько атрибутов, которые записываются подряд. При этом порядок их указания не играет роли:

<p class="text-center" title="Этот элемент имеет несколько атрибутов">Параграф с несколькими атрибутами</p>
