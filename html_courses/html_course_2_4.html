HTML: Списки

Списки — удобный способ отображения информации на странице. Они позволяют легко объединять связанные элементы, например, список покупок:

— Молоко
— Хлеб
— Вода

Можно сделать такой список с помощью трёх параграфов, но тогда, с первого взгляда, будет казаться, что это три разные фразы не объединённые общим смыслом.

Молоко
Хлеб
Вода

Для правильного объединения такой информации применяются списки. В HTML они создаются с использованием нескольких вложенных тегов:

  • <ul> или <ol> — для указания типа списка.
  • <li> — для добавления элемента списка.

Пример базовой структуры списка выглядит так:

<тип_списка>
    <пункт_списка> Текст </пункт_списка>
</тип_списка>

Маркированный список

Для создания маркированного списка используется тег <ul>. Он обозначает, что элементы списка будут отображаться с маркерами. Например:

<ul>
    <li>Молоко</li>
    <li>Хлеб</li>
    <li>Вода</li>
</ul>
  • Молоко
  • Хлеб
  • Вода

Нумерованный список

Для упорядочивания информации применяются нумерованные списки, где каждый элемент имеет свой порядковый номер.

Такие списки особенно полезны для обозначения последовательности шагов или действий. Нумерованный список создаётся с использованием тега <ol>, внутри которого элементы размещаются в тегах <li>.

В разметке HTML выглядит следующим образом:

<ol>
    <li>Купить корм</li>
    <li>Сходить к Алексею</li>
    <li>Приготовить ужин</li>
</ol>

Список дел на день

  1. Купить корм
  2. Сходить к Алексею
  3. Приготовить ужин
